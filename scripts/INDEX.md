# scripts

可执行脚本集合：通过 869 HTTP API 发送纯文本与媒体消息。

## Files

| File | Role | Function |
|------|------|----------|
| send_869_text.py | Exec | 发送纯文本消息，支持直接传文本或从文件读取，并可附带 AtWxIDList |
| send_869_media.py | Exec | 发送图片/视频/语音/链接/文件（“音乐”=语音别名） |
